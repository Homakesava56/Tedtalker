# The Boyz AI Speaker Bot

A GitHub Pages-ready interactive speaker booking chatbot.

## Features

- Branded banner image
- Topic selection
- Speaker search by full or partial name
- Full speaker topic list
- Interactive one-question-at-a-time booking
- Budget comparison against minimum speaker fees
- Affordable alternative recommendations
- Funny responses to out-of-scope questions
- Mobile-friendly interface

## Files

- `index.html`
- `style.css`
- `script.js`
- `speakers.json`
- `assets/the-boyz-banner.png`

## Deploy on GitHub Pages

1. Create a new public GitHub repository.
2. Upload all project files and the `assets` folder.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose `main` and `/ (root)`.
6. Save and wait for deployment.

Your URL will be:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`
